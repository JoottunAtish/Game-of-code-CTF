Bubble Sort Explained for Kids!

Imagine you have a bunch of numbered toys, and you want to arrange them in order from the smallest to the biggest number. Bubble sort is like a game where you compare pairs of toys and swap them if they are in the wrong order. You keep playing this game until all the toys are in the correct order!

Bubble sort is one of the easiest sorting algorithm to learn. There are more that are complex and more fast.

            // Psuedocode

            FUNCTION bubble_sort(toys) RETURNS ARRAY
                DECLARE n : INTEGER = LENGTH(toys)
                DECLRE temp: INTEGER

                FOR i  = 0 TO n-1
                    FOR j = 0 TO (n -1 - i)
                        IF (toys[j] > toys[j + 1]) THEN
                            temp = toys[j]
                            toys[j] = toys[j+1]
                            toys[j+1] = temp
                        ENDIF
                    ENDFOR
                ENDFOR
                RETURN toys
            ENDFUNCTION


            # Python
            def bubble_sort(toys):
                n = len(toys)
                
                for i in range(n):
                    for j in range(0, n-i-1):
                        if toys[j] > toys[j+1]:
                            temp = toys[j]
                            toys[j] = toys[j+1]
                            toys[j+1] = temp

                return list(toys)
            
            # Example usage
            toys = [4, 2, 7, 1, 3]
            sorted_toys = bubble_sort(toys)

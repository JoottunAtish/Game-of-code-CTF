Iteration and Looping

Iteration statements are like your friendly guide that helps you repeat tasks.

Loops are like magical spells that can make things happen repeatedly based on certain conditions. They help us perform tasks multiple times, just like when you keep turning objects into apples or bananas with your magic wand. So, loops are super important because they help us automate tasks and do cool things over and over!

Have you ever needed to do something repeatedly? Let's explore the power of loops:

Imagine you have a magic wand that can transform items into apples or bananas repeatedly!

If I feel a little happy, I will eat apples. But if I am very, very happy, then I will eat bananas.

Pseudocode for the task above using loops:

        FOR i <-- 1 to 5:
            OUTPUT i
        ENDFOR

in the code above, the i is a variable that is incremented each time output is done. it stops when the value becomes 5. so it will display (1,2,3,4,5)

In Python code with loops, we would write:

        
            for i in range(5):
                print(i)

Looping Structures

There are different types of loops like the FOR loop(Count control), WHILE loop(Pre-conditional looping), and DO-WHILE loop(Post-conditional Looping) that help us repeat tasks efficiently.

    • FOR LOOP -- Count Control Looping
      WHILE LOOP -- Pre Conditional Looping
      REPEAT LOOP or DO-WHILE LOOP -- Post Conditional Looping

Intro to coding!

In this journey to learn to code, we will teach you how to where to start and how to code. We will start with the fundamental like Psuedocode and teach you a programming language called python. Additionally we will also teach you the basics of logic.

Difference between Psuedocode and a programming language

Psuedocode is not a programming language, rather it is used in the design process to show how the code will look and if the logic is correct. it is written in paper. Then if the logic of program is correct, we can then transfer to a programming language like python.
python is a high level programming language, meaning the program is closer to the english language. Python is also case-sensitive. Meaning, `Print` and `print` are different.

Lets get started then!

Display of messages

Why do we need to make display? It is simple. It is to provide feedback to the user. Example, lets say you are downloading a game or a program, it shows you a progress bar and how much of the program has been downloaded. It gives you a visual respond and it interacts with the user.

Lets make the program to display 'Hello, world!!'

        
            Psuedocode:
                OUTPUT "Hello, world!!"

            Python:
                print("Hello, world!!")

Declaration of variables
